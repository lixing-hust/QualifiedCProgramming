# Hello String

## 问题描述

实现一个函数，接收一个字符数组 `out`，将其内容改写为固定字符串 `"hello"`。

约定：
- 调用者提供长度至少为 `6` 的可写字符缓冲区。
- 函数本身不分配堆内存。
- 函数会向 `out` 依次写入 5 个字符：`h`、`e`、`l`、`l`、`o`。
- 最后一个位置写入字符串结束符 `'\0'`。

## 正确代码

```c
void hello(char *out) {
    out[0] = 'h';
    out[1] = 'e';
    out[2] = 'l';
    out[3] = 'l';
    out[4] = 'o';
    out[5] = '\0';
}
```

## 说明

这段代码不再返回只读字符串字面量，而是直接修改调用者传入的缓冲区：
- `out[0..4]` 被写成 `"hello"`
- `out[5]` 被写成字符串结束符 `'\0'`
- 函数结束后，`out` 表示一个标准的 C 风格字符串 `"hello"`

因此函数的效果始终是把输入缓冲区改写为字符串 `"hello"`。
